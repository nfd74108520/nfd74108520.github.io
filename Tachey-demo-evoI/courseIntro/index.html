你可以學到 //

1. 知道感覺統合包含哪七個感覺，以及它的理論元素
2. 知道感覺統合對孩子的發展影響，以及注意孩子的表現對應的感覺統合問題
3. 掌握適當的感覺統合遊戲操作方式，給孩子最恰到好處的刺激
4. 透過適量感覺統合刺激，幫助孩子發展專注力、學習力、情緒、動作、自理、人際關係

課程詳情 //

常覺得孩子難搞？我們可以從「感覺統合」來改善

您是否記得這世界曾經如此陌生？

還記得超級英雄剛得到超能力的樣子嗎？總是不懂得控制自己——跑得太快，跳得太高，把東西拉壞。看得太清楚、聽得太敏銳，覺得這個世界好亮好吵，感到害怕，想把感覺關掉卻做不到，直到訓練好一陣子才漸漸適應自己的超能力。 其實，0 到 2 歲的孩子剛來到這個世界上，也像忽然得到超能力一樣，一邊急速成長發育，一邊認識這個世界；對於色彩、氣味、聲音，都感到陌生，對於溫度、觸感都覺得不熟悉，甚至覺得太過刺激。所以有些孩子，容易分心、反應慢，莫名其妙的逃避或哭鬧，踩在草地或沙地上就不舒服、穿上某些衣服就扭來扭去、新衣服一定要把標籤剪掉，總是跌跌撞撞、動作魯莽、極度討厭肢體碰觸。這些，都可能是感覺統合失調所造成的。
